# Gemini Code Assistant Context

## Project Overview

This is a simple web application built with Vite. It features a basic counter functionality and serves as a starting point for a web project. The application is written in vanilla JavaScript.

## Building and Running

To work with this project, you need to have Node.js and npm (or a compatible package manager) installed.

### Installation

To install the project dependencies, run the following command in your terminal:

```bash
npm install
```

### Development Server

To start the development server, run:

```bash
npm run dev
```

This will start a local server, and you can view the application in your web browser at the address provided in the terminal output.

### Building for Production

To build the application for production, run:

```bash
npm run build
```

This will create a `dist` directory with the optimized and minified files ready for deployment.

### Previewing the Production Build

To preview the production build locally, run:

```bash
npm run preview
```

## Development Conventions

*   **Framework:** This project uses Vite for the build tooling and development server.
*   **Language:** The application is written in vanilla JavaScript.
*   **Styling:** Basic styling is provided in `src/style.css`.
*   **Code Structure:**
    *   `index.html`: The main HTML entry point.
    *   `src/main.js`: The main JavaScript entry point.
    *   `src/counter.js`: Contains the counter logic.
    *   `public/`: Contains static assets.
